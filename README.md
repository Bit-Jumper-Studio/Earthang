# Sols

**Sols** is a group of development tools focused on performance, low-level control, and native binaries.

The project brings together compilers, languages, and frameworks designed to give developers full control over how their software is built and executed.

## Compilers

- **Earthang** `[X]`  
  A programming language and compiler for Linux

- **C+Compiler** `[ ]`  
  A C/C++ compiler focused on simplicity and native code generation.

- **HTJS** `[ ]`  
  A JavaScript-to-native compiler based on Node.js.
